# Scripts

This directory contains all R scripts used in the manuscript.

| Script | Description |
|---------|-------------|
|01_IMD_trend_analysis.R|Trend analysis, ETCCDI indices, Mann–Kendall, Pettitt, SPI and Hurst analysis|
|02_ERA5_circulation.R|ERA5 circulation and moisture-flux analysis|
|03_CMIP6_ensemble_extraction.R|CMIP6 model processing and ensemble extraction|
|04_Attribution_analysis.R|Dynamic and thermodynamic attribution analyses|
|05_Context_map.R|Study area and context map generation|
