# Data

This repository does not redistribute proprietary datasets.

The analyses require the following datasets:

- India Meteorological Department (IMD) daily gridded rainfall
- ERA5 reanalysis (Copernicus Climate Data Store)
- CMIP6 historical simulations (Earth System Grid Federation)

Users should obtain these datasets directly from the respective data providers before reproducing the analyses.

This directory is intended to store locally downloaded datasets and is intentionally left empty.
